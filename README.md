 Tech News Finder

A lightweight, high-performance web application designed to fetch, display, and filter the latest trending stories in technology and software development. 

This project demonstrates core computer science engineering concepts, including asynchronous programming, third-party API integration, and dynamic user interface state management without reliance on heavy external frameworks.

Live Demo
https://github.com/Syedwajeeh5/Tech-News

Key Features
*Real-Time Data Fetching:** Integrates with the official Hacker News API to stream live tech articles.
*Dynamic API Endpoints:** Allows users to switch instantly between 'Top', 'New', and 'Best' trending categories.
*Instant Client-Side Filtering:** An optimized search engine that filters article titles dynamically as the user types.
*State-Driven Theme Engine:** A custom-coded Dark Mode toggle that adjusts UI elements seamlessly via CSS variables.

 Technical Skills Demonstrated
*Languages:** JavaScript (ES6+), HTML5, CSS3
*Asynchronous JS:** Promises, `async/await`, and the `fetch()` API
*DOM Manipulation:** Dynamically creating, appending, and filtering UI elements
*Responsive Design:** Mobile-first layout grids optimized for all screen sizes
